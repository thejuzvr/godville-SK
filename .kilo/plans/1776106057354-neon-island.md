## 2. Храм Даэдра (/temple)

### Цель
Реализовать систему храма с постройкой, бонусами, NPC-постояльцами и врагами.

### Backend (lib/godville_sk/)

#### 2.1 Расширение HeroTemple (game/hero_temple.ex)
```elixir
field :construction_progress, :integer, default: 0       # 0-100%
field :total_invested, :integer, default: 0              # Всего вложено (5000 = 100%)
field :temple_level, :integer, default: 0                # Уровень храма
field :donations_count, :integer, default: 0             # Количество пожертвований

# NPC постояльцы (автоматически по порогу)
field :residents, {:array, :string}, default: []         # Текущие NPC

# Враги
field :enemies, {:array, :string}, default: []           # Активные враги
field :defense_level, :integer, default: 1               # Уровень защиты
field :enemy_encounters, :integer, default: 0            # Всего атак

# История событий
field :temple_events, {:array, :map}, default: []        # События храма
```

#### 2.2 Создание TempleMechanics (game/temple_mechanics.ex)
- `blessings_for_progress(progress)` - бонусы по % постройки
- `auto_spawn_residents(progress)` - авто-призыв NPC по порогу
- `maybe_spawn_enemy(tick_count, progress)` - враг атакует (каждые 100 тиков)
- `apply_temple_bonuses(hero)` - применить бонусы к герою

#### 2.3 Интеграция с StateMachine
- Добавить обработку пожертвований ( donate_gold )
- Добавить проверку врагов (каждые 100 тиков)
- Обновлять бонусы в tick()
