# План: Настройка профилей тиков в админ-панели

## Текущее состояние
- В `hero.ex:192-217` уже есть динамический расчёт интервала тика в зависимости от статуса героя
- Админ-панель (`admin_live.ex:191-226`) позволяет только установить фиксированный интервал
- Применение требует перезапуск сервера

## Задача
Добавить в админ-панель выбор предустановленных профилей для ускорения/замедления тиков без перезапуска.

## План реализации

### 1. Создать модуль профилей тиков (lib/godville_sk/tick_profile.ex)
- Определить профили: slow, normal, fast, turbo
- Каждый профиль содержит карту интервалов для каждого статуса
- Функции для получения профиля и расчёта интервала

### 2. Обновить hero.ex
- Интегрировать TickProfile в `calculate_tick_interval/1`
- Использовать активный профиль для расчёта интервалов

### 3. Расширить админ-панель (admin_live.ex)
- Добавить секцию "Tick_Profile_Selector" с кнопками выбора профиля
- Показывать текущий активный профиль
- Кнопки: Slow (замедление), Normal (по умолчанию), Fast (ускорение), Turbo (макс. скорость)

### 4. Обновить handle_event для смены профиля
- Добавить событие "set_tick_profile"
- Сохранять профиль в stateGenServer
- Применять без перезапуска

## Профили (предложение)

| Профиль   | idle (ms)     | combat (ms) | questing (ms) | resting (ms) | trading (ms) |

| Normal    | 5000-120000   | 2300        | 60000-300000  | 3000-8000    | 5000-15000   |
| Fast      | 1000-30000    | 1000        | 15000-60000   | 1000-3000    | 2000-5000    |
| Turbo     | 500-5000      | 500         | 5000-15000    | 500-1500     | 1000-2000    |
